==STEPS TO RUN THE PROJECT

2. Make sure XAMPP is installed on your machine
3. Move this project folder to C:\\xampp\\htdocs
5. Open XAMPP, then start Apache and MySQL server
6. Open the project in browser at localhost/Aegis
